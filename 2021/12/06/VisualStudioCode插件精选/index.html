配置推荐

按键推荐

  1. 在 windows 上，将默认的触发提示的键 “ctrl+space” 改为 “alt+/”；
  2. 修改 gitlens 的快捷键 “alt+/” 为 “ctrl+shift+alt+/”，避免冲突；
  3. 解决 windows 上和 Path Intellisense 的按键 BUG，添加 { "key": ".", "command": "" }。
